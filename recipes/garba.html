Garba

Description

Un plat ivoirien largment consommé sur le continent africain.

Très doux et rassasiant, ce plat est un véritable régal pour les amateurs de cuisine africaine.

Ingrédients

  • Attieké
  • Poisson Thon
  • Piments frais
  • tomate
  • Oignon
  • Farine
  • De l'assaisonnement

Étapes

  1. Renverser de la farine dans un bol
  2. Ajouter le poisson thon dans la farine et melangé
  3. Faire frire le poisson thon dans une poêle
  4. Verser de l'Attieké dans une assiette
  5. Ajouter le poisson thon frit sur l'Attieké
  6. Ajouter les piments frais, les tomates et les oignons
  7. Ajouter de l'assaisonnement
